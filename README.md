## Requirements

Ruby MRI 2.5.0+

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'on_strum-logs'
```

And then execute:

```bash
bundle
```

Or install it yourself as:

```bash
gem install on_strum-logs
```
